What are these predictions?

A Temporal Fusion Transformer (TFT) machine learning model analyses historical sales patterns to predict the expected daily demand for each drug class over the next 7 days. The inventory rule engine uses these forecasts to automatically raise Reorder alerts before stock runs out. Predictions are generated offline and loaded into the database — they reflect the most recent model run.
