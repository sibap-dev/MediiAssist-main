# 🏥 MediAssist

A full-stack medical inventory management web application developed to streamline medicine stock handling, inventory tracking, and product management through a clean and responsive interface.

🔗 **GitHub Repository:** https://github.com/sibap-dev/MediiAssist-main

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **Database:** PostgreSQL
- **Version Control:** Git & GitHub

---

## 🚀 Features

- 💊 Add, Update, and Delete Medical Inventory Records  
- 📦 Real-Time Medicine Stock Management  
- 🔍 Search and Track Inventory Details Efficiently  
- 🗂️ Organized Product and Medicine Information System  
- 📱 Responsive User Interface for Better Accessibility  
- 🔐 Backend Integration for Secure Data Handling  
- ⚡ CRUD Operations using Flask and PostgreSQL  

---

## 📚 Project Overview

MediAssist was developed as a laboratory project focused on building a practical medical inventory management solution. The application helps manage medicine records efficiently while improving understanding of full-stack web development concepts such as backend integration, database management, and CRUD operations.

---

## 🧠 Learning Outcomes

- Developed hands-on experience with Flask backend development  
- Improved understanding of PostgreSQL database operations  
- Implemented CRUD functionalities in a real-world project  
- Enhanced UI structuring and responsive web design skills  
- Learned integration between frontend, backend, and databases
